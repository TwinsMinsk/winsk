# Создание терминальных глитч-анимаций

## Use this skill when
- Пользователь просит анимировать текст на главной странице (Hero-секция).
- Нужно создать эффект "хакерского терминала" или декодинга (например, перестроение слова Minsk в Winsk).
- Требуется добавить микро-взаимодействия для Bento-box UI.

## Do not use this skill when
- Нужно анимировать простые переходы между страницами (page routing).
- Задача касается базовых CSS-транзиций (hover, focus).

## Instructions
1. Используй Framer Motion для управления стейтом анимации, но саму логику перебора символов (scrambling) пиши через кастомный React-хук (`useScrambleText` или `useTypewriter`), завязанный на `requestAnimationFrame` для максимальной производительности.
2. Используй моноширинный шрифт (JetBrains Mono) для всех анимируемых терминальных блоков.
3. Добавляй префикс `> ` и мигающий курсор `_` в конце строки для имитации командной строки.
4. Убедись, что компонент имеет директиву `'use client'`, так как глитч-эффект требует работы с DOM и `useEffect`.
