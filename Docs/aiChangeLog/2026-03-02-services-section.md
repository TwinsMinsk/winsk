# Добавлен блок "ServicesSection" (Услуги)

**Date:** 2026-03-02
**Description:** Сгенерированы кинематографические AI-изображения и создан блок услуг с Cyberpunk Glassmorphism карточками.

## Что было сделано:
1. Сгенерированы 4 футуристичных изображения (промптирование, 16:9) для иллюстрации предлагаемых услуг:
   - `service-mvp.jpg` (Архитектура ПО, голография)
   - `service-bots.jpg` (Нейросети, ИИ, дата-потоки)
   - `service-automation.jpg` (Автоматизация, оптоволокно)
   - `service-web.jpg` (Стильный веб-интерфейс, неон)
2. Изображения сохранены в `public/services/`.
3. Разработан клиентский UI-компонент `ServicesSection.tsx`:
   - Сетка `grid-cols-1 md:grid-cols-2`.
   - Использован `<Image>` для фоновых изображений (с `quality=90` и `object-cover`).
   - Настроен сложный градиентный оверлей.
   - Снизу карточки реализована панель матового стекла (backdrop-blur-md) для текстов и тегов.
   - Добавлены анимации Framer Motion: появление секции (`whileInView`) по очереди (stagger), увеличение фоновой картинки (`scale: 1.05`) и появление светящейся обводки карточки по `whileHover`.
4. Секция интегрирована в дизайн `app/page.tsx` сразу после `Hero`.
5. Исправлено предупреждение `tsc` с типизацией анимационных `Variants` у `framer-motion`.

## Проверочные шаги (Verification):
- `npx tsc --noEmit` — успешно. Дизайн можно проверить на странице браузера локального сервера.

## Отклонения:
Нет. Строгое соответствие техническому заданию спецификации "Cinematic Glassmorphism".
