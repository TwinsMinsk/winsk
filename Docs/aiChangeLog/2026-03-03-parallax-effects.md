# Parallax Effects & Grid Dynamics

## Описание
В рамках развития визуального языка и UX ("Wow Factor"), интегрирован Parallax эффект для тяжелых дата-сеточных компонентов. 

## Детали реализации:
1. **BentoSkills.tsx**
   - Установлена зависимость от `useScroll` и `useTransform` из `framer-motion`.
   - Внедрен scroll offset-провайдер с референсом на корневой `section`.
   - Каждая из 4-х карточек профиля получила свой уникальный физический вес по оси Y (y1: [-40, 40], y2: [-80, 80], и т.д.).
   - При скролле карточки разлетаются в перспективе глубины.

2. **ServicesSection.tsx**
   - Добавлен независимый параллакс-сдвиг для 4-х кинематографичных карточек услуг (yOffsets от 30 до 100).
   - Исправлена опечатка закрывающего тега `</div>` в стеклянном контейнере текста.

3. **Верификация**
   - `npm install framer-motion` (т.к. мы обновили версию хуков).
   - Запуск `npx tsc --noEmit` успешно завершен (`Exit code: 0`).
